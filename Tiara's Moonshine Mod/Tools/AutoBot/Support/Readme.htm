Basic Instructions

Note: AutoHotKey may need to be installed for some computers (http://www.autohotkey.com/download/)

 

1)  FOR XP USERS: Double Click Fossil Restoration 10.1.exe in the AutoHotKey folder.

     FOR VISTA/7 USERS: Right Click Fossil Restoration 10.1.exe in the AutoHotKey folder and choose "Run as Administrator"

 

2) Switch to Mabinogi

 

3) FOR MANUAL CONTROL: Simply use a fossil in your inventory and press the "Numpad Enter" key or the "X" key.

FOR AUTOMATIC RESTORATION: Press the "Z" or "A" (for au3 version) key. This will turn on Autodetect. Press Z or A (for au3 version) again to turn it off.

 

*If on manual control, repeat step 3 for each fossil.

*If Auto Detect is on, simply "use" each fossil in your inventory, then press "Z" or "A" to turn it off.

*If the window is detected, a sound will be played. If not, resort to the FAQ section.*

 

Other commands:

Pause - End (Pauses the script only, it will resume where it left off)

Insert - Suspend (Use this if your regular in-game bind are numpads, it will allow you to use them again until unsuspended)

Ctrl+R - Reloads the script. It halts the whole script completely.

Exit Program - Ctrl+Esc

 

Things to keep in mind:

 

·    Do not move the mouse during the automatic recover process. Moving the mouse will most likely damage your fossil.

·    If you did move the mouse or badly damaged the fossil, switch channels or log out (type /logout in the chatbox) to save the fossil.

·    Don't forget to turn the fossil into Belita.

 

 

Troubleshooting:

 

Q:Your program does nothing at all

A: 1) If you're running window vista, you will need to right click the application and run as administrator. If you're on Window XP, try re-downloading the macro.

    2) Make sure all necessary files are in place.

 

Q:Your program is working, but it's very choppy and doesn't cover all the sulfur

A:This is the result of your game is lagging. Lagging will cause the script to skip a few spots, and will most likely not even choose the correct brush size. Reduce your speed (See "Changing Restore Speed" below) If that doesn't help, go into Simon's shop or a nearby building to reduce lag. If it still doesn't help, you'll need a better computer.

 

Q:I can't get a perfectly restore on some fossils

A:The program doesn't cover ALL the sulfur for every fossils, but about 99.995% of them. The others are perfect. We are working on this.

 

Q:The program cannot detect the image!

A:This shouldn't happen.

1)Make sure you have the correct image files are in the Auto Hotkey\Find\Images\ folder.

2)Remove anything that is obstructing the fossil window (Including the mouse).

3)You may need to use the 64 bit version.

 

Advanced Operations:

Changing Hotkeys

 

You will need AutoHotKey installed. In the Source directory, edit the autohotkey file. Replace the hotkeys (For example CTRL-Escape is ^Esc∷    See AutoHotKey documentation for details). Then recompile the program and replace the original .exe with your new one.

 

Changing Restore Speed

for ahk edit the speed.ini. The higher the value, the slower it goes. Save the file and reload the macro.

for au3 edit the config.ini. The higher the mousedelay value, the slower it goes. Save the file and reload the macro.

 

Updating an RST file

Simply drag the updated RST file into the Auto Fossil Restoration/Find/Macros folder.

 

Adding a new fossil

Simply drag a PNG image of the fossil into the Fossil Restoration/Find/Images folder and a corresponding (same name) RST file into the Fossil Restoration/Find/Macros folder
The old days of editing thousands of mouse actions in the main program have come to an end!
Each individual fossil has its own individual file containing the restore data (extension .RST). Its a simple code that goes like this:

! = Click mouse
D = Mouse Down
U = Mouse Up
X,Y = Mouse coordinates (replace X and Y with X value and Y value, respectively)
Any other characters are ignored. Meaning you can leave comments inside the file itself!

It makes editing and redistributing restore data so much easier! In fact, all you do is change/add a value, hit save, and its ready to go! To redistribute it, just put the file up for download. End Users replace their version, and now they have your change too!
